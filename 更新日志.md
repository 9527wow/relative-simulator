## 📝 使用方法

### 打开设置
1. 点击右上角的 ⚙️ 齿轮图标
2. 设置面板从右侧滑出
3. 修改设置后自动保存
4. 点击 × 或点击外部区域关闭

### 切换深色模式
1. 打开设置面板
2. 在"主题模式"中选择"🌙 深色模式"
3. 界面立即切换到深色主题
4. 设置自动保存

### 自定义对话轮数
1. 打开设置面板
2. 在"对话设置"中选择"最大对话轮数"
3. 选择你想要的轮数（5/10/15/20/无限）
4. 开始游戏时生效

### 添加音效
1. 将音效文件放入 `sounds/` 文件夹
2. 确保文件名正确（send.mp3, receive.mp3 等）
3. 在设置中开启"音效反馈"开关
4. 音效自动播放
