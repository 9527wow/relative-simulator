# 🔊 音效文件说明

此文件夹用于存放应用的各种音效文件。

## 需要的音效文件

### 1. 发送消息音效
- **文件名**: `send.mp3`
- **格式**: MP3
- **时长**: < 1秒
- **描述**: 清脆的发送音效

### 2. 接收消息音效
- **文件名**: `receive.mp3`
- **格式**: MP3
- **时长**: < 1秒
- **描述**: 温柔的提示音

### 3. 点击音效
- **文件名**: `click.mp3`
- **格式**: MP3
- **时长**: < 0.5秒
- **描述**: 轻快的点击音

### 4. 胜利音效
- **文件名**: `win.mp3`
- **格式**: MP3
- **时长**: 2-3秒
- **描述**: 欢快的胜利音效

### 5. 错误音效
- **文件名**: `error.mp3`
- **格式**: MP3
- **时长**: < 1秒
- **描述**: 温和的错误提示音

## 如何添加音效

1. 将音效文件放入此文件夹
2. 确保文件名与上述列表匹配
3. 在设置中开启"音效反馈"开关
4. 音效将自动播放

## 免费音效资源

- [Freesound](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [Mixkit](https://mixkit.co/free-sound-effects/)

## 注意事项

- 请使用免费或拥有授权的音效
- 音效文件不宜过大（建议每个文件 < 100KB）
- 音量应适中，不要过大或过小
- 确保音效格式兼容（推荐 MP3）
