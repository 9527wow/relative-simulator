# 🎉 亲戚应对模拟器 - 使用说明

## 🚀 快速开始

### 第一步：打开应用

直接双击 `index.html` 文件，应用会在你的默认浏览器中打开。

### 第二步：选择难度

应用会显示4个难度等级的亲戚角色：

| 角色 | 难度 | 特点 |
|------|------|------|
| 👩 和蔼的大姨 | ⭐ 简单 | 性格温和，问题简单，适合新手 |
| 👩‍🦳 犀利的大姑 | ⭐⭐⭐ 中等 | 言辞犀利，擅长对比施压 |
| 👨 王牌二舅 | ⭐⭐⭐⭐⭐ 困难 | 经验丰富，问题刁钻层层递进 |
| 👵 终极三婶 | ⭐⭐⭐⭐⭐⭐ 极难 | 全方位无死角轰炸，究极挑战 |

### 第三步：开始对话

1. 点击任意角色卡片开始挑战
2. 亲戚会主动抛出一个刁难问题
3. 在输入框中输入你的应对策略
4. 点击"发送"按钮或按 Enter 键
5. 查看 AI 的犀利回复

### 第四步：挑战结果

- 每完成一轮对话计为1轮
- 达到10轮后挑战结束
- 根据轮数评价你的应对能力

### 🎮 使用技巧

1. **从简单开始**：建议先从"和蔼的大姨"开始练习
2. **保持礼貌**：虽然是应对，但建议保持基本的礼貌
3. **灵活应对**：可以使用幽默、转移话题、适度拒绝等策略
4. **观察规律**：注意不同角色的提问模式，找到应对方法

## 💡 常见问题

### Q: AI 回复很慢怎么办？
A: 这是正常现象，API 调用需要时间，请耐心等待加载动画完成。

### Q: 可以更换亲戚角色吗？
A: 可以，点击聊天界面左上角的"← 返回"按钮即可重新选择。

### Q: 对话记录会保存吗？
A: 目前不会保存，刷新页面后会重新开始。

### Q: API Key 会泄露吗？
A: API Key 在前端代码中，建议仅用于个人测试。如需公开分享，请考虑添加后端代理。
